Roles & Permissions

Create roles, assign them to users by name, and view your own permissions.

Create Role — POST /roles/create
Assign Role to User — POST /users/assign-role
View User Roles / Permissions

Documents

Upload financial documents. Each upload is automatically split into chunks and indexed into ChromaDB for semantic search.

Upload — POST /documents/upload
All Documents — GET /documents
Get / Delete — GET|DELETE /documents/{id}

Metadata Search

Filter documents by company name or type using a plain SQL query. For meaning-based search, use Semantic Search.

GET /documents/search

Semantic Search

Ask a question in plain English. Finds relevant document chunks by meaning — not just keyword matching.

Query→ Embed→ Top 20 chunks→ CrossEncoder rerank→ Top K results
POST /rag/search

View Chunks

Inspect what ChromaDB has stored for a document — all chunks in order.

GET /rag/context/{document_id}

Manage Embeddings

Re-index a document if content changed, or remove its embeddings from ChromaDB without deleting the document.

Re-index — POST /rag/index-document
Remove Embeddings — DELETE /rag/remove-document/{id}
